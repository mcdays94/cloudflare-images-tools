# Icon options

Four candidate icons explored during the v0.5 polish phase. The
**Stack & Cloud** option was chosen and promoted to
`packages/raycast/assets/icon.png`.

| File | Concept | Status |
|---|---|---|
| `01-stack-cloud.png` / `.svg` | Two stacked photo cards with a soft cloud floating above | ✅ Chosen |
| `02-beam-up.png` / `.svg` | Photo card emitting an upward beam into a cloud at the top | runner-up |
| `03-gallery-grid.png` / `.svg` | 2×2 grid of image tiles (mountain, cloud, horizon, upload arrow) | considered |
| `04-camera-cloud.png` / `.svg` | Camera silhouette with a cloud where the lens aperture would be | considered |
| `compare.png` | All four side-by-side at 256×256 each, for at-a-glance comparison | reference |

The SVG sources are kept here in case you want to:

- Refine the chosen concept in Illustrator / Figma / Photoshop
- Iterate on a runner-up later
- Reuse a colour palette in marketing copy / blog posts
- Recall the design rationale (each SVG has a header comment explaining
  its concept, colours, and intended reading)
